# 📦 Inventory Management App

A web-based application to manage your shop/company inventory efficiently.

---

## 🚀 Features

- 📊 **Dashboard**
  - View insights about products, pricing, and low-stock alerts

- 📋 **Inventory Table**
  - See all products with details like price, quantity, and more

- 🔍 **Search & Sort**
  - Search products by name
  - Sort by:
    - Name
    - Price
    - Quantity
    - Low stock

- ➕➖ **Manage Products**
  - Add new products
  - Delete existing ones

---

## 🛠️ Tech Stack

- ⚡ **Next.js**
- 🟦 **TypeScript**
- 🎨 **Tailwind CSS**
- 🔐 **Stack Auth**
- 🐘 **Neon PostgreSQL**
